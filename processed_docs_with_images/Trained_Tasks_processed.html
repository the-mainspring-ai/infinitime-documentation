Trained Tasks

Trained Tasks allow the user to set up specific Task names for a job assigned to an employee. Employees who are qualified to perform a specific task should have each task they are trained to perform assigned to them.

 

Assigning Trained Tasks to an Employee:

 

  1. Click Insert

 

 

  1. Use the magnify glass to select a trained task to assign to the employee, or type the first few letters of the trained task and it will automatically be filled in with auto-complete.

 

 

 

  1. Enter the wage for the corresponding trained task.
