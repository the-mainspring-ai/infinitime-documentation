Reader Settings - Relay Settings

The Thor Terminal includes two dry contact relays accessible from the rear terminal. Relay One includes connections labeled COM1, NC1, and NO1 while Relay Two includes connections labeled COM2, NC2, and NO2.

If InfiniTime is licensed for both Bells and Access Control the Relay Settings tab can be used to set the function of the individual relays. If InfiniTime is not licensed for both Bells and Access Control, Relay 1 will be configured for Bells by default and the Relay 2 setting will not be available.
