Juno Reader Setup - USB Direct Connect

After connecting the Juno to the InfiniTime server and ensuring the menu settings are configured properly, follow the steps below to connect to the Juno from the InfiniTime Application.

 

Adding a USB Direct Connect Juno to the InfiniTime Software

 


  1. Click on Lookups

 

 

  1. Click on Reader Configuration

 

  1. Click on Insert

 

 

  1. Enter a description for the Juno Reader in the Port Name Field. This description is displayed in the Reader Configuration Table.

 

 

  1. Select Juno from the Type Drop Down.

 

 

  1. Select the COM Port associated with the USB to Serial Converter, as identified using the Determining COM Port for Juno Direct Connection section of this document.

 

 

  1. Ensure 115200 Baud is selected.

 

 

  1. Click on the Reader Address Tab.

 

 

  1. Click Insert.

 

 

  1. Enter a description for the Reader Address in the Description Field. This description is displayed in the System Monitor.

 

 

  1. Ensure the Address is set to 1.

 

 

  1. Click on the Polling Tab and set the desired polling interval.

 

 

  1. Uncheck the Inactive Check Box.

 

 

  1. Click OK to save the Reader Address.

 

 

  1. Click OK to save the Reader Type.

 

 

  1. Click Close to close the Reader Configuration Table.

 

 

  1. Follow the Steps to Update the Juno Reader.
